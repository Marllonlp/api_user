# API CRUD Simples de Usuários (Django)

Este é um projeto simples que demonstra uma API CRUD (Create, Read, Update, Delete) para gerenciar usuários, construída com Django e Django Rest Framework.

## Funcionalidades

* Criar novos usuários.
* Listar todos os usuários.
* Obter detalhes de um usuário específico (pelo nickname).
* Atualizar um usuário existente.
* Excluir um usuário.
